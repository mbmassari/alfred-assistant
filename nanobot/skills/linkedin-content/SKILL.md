# Skill: LinkedIn Content

Cria posts e artigos para LinkedIn com tom profissional.

## Quando usar
- Usuário pede para criar conteúdo para LinkedIn
- Usuário quer compartilhar insights, experiências ou conhecimento técnico

## Processo

1. **Consultar template** — Leia `templates/linkedin-artigo.md` no vault
2. **Consultar contexto** — Verifique `knowledge-base/` para dados do perfil profissional
3. **Gerar conteúdo:**
   - Hook (primeira linha que aparece no feed — essencial para engajamento)
   - Corpo com parágrafos curtos (2-3 linhas cada)
   - CTA final
   - 3-5 hashtags relevantes
4. **Apresentar para aprovação** — Mostre o draft e aguarde confirmação
5. **Salvar no vault** — Após aprovação, salve em `notes/linkedin-YYYY-MM-DD-{tema}.md`

## Regras
- Tom profissional com personalidade (ver USER.md)
- Parágrafos curtos para escaneabilidade mobile
- Máximo 5 hashtags
- NUNCA publicar sem aprovação explícita
- Use line breaks generosos

## Formatos

### Post curto (feed)
- 100-300 palavras
- Ideal para insights rápidos, opiniões, dicas

### Artigo longo
- 500-1500 palavras
- Ideal para análises, tutoriais, cases
- Incluir subtítulos e bullet points
